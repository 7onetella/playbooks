Usage: deploy.sh 
```
deloy.sh 
```


## Why
I needed to set up a local lab environment. Following services are created using the playbooks in this repo.

- ApacheDS
- Consul server & agent
- Docker registry
- Fabio
- Go install
- Grafana
- Graphite
- Java8 install
- Java11 install
- Jenkins
- Jupyterlab
- K3S
- Neo4j
- Nomad cluster & agent
- Nutcracker aka twemproxy
- Postgres
- PgAdmin
- Prometheus
- Redis
- Vim-go
